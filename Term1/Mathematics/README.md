
# Mathematics 1 (ریاضی عمومی ۱)

### Application in Programming
Although calculus is not directly used in frontend development, it builds strong analytical thinking.  
It can be useful in:
- Animations (e.g., using sine and cosine for smooth movements)
- Game development or physics-based effects
- Understanding curves, motion, and time-based behaviors in UI

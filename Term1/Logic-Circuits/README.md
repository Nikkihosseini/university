# Logic Circuits (مدارهای منطقی)

### Application in Programming
Logic Circuits builds the foundation of how computers work.  
In programming, it’s useful for:
- Understanding Boolean logic (AND, OR, NOT)
- Writing clean conditional code
- Thinking like a computer (bitwise logic, control flow)
